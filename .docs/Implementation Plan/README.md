# Implementation Plan

This folder contains the MVP sprint plan and epic breakdown for Nudgly. Epics are kept small in scope so each can be completed and validated (FE + BE) before moving on.

**References:**
- [Functional & technical design (app-idea)](../Design%20Docs/app-idea.md) — MVP scope, schema, API, nudge engine, §16–§17.
- [UI/UX design](../Design%20Docs/ui-ux-design.md) — Layout, screens, theme system, interactions.

**Principle:** Each epic delivers both a backend and a frontend slice so that at completion the functionality can be tested end-to-end.
